# Texas Hold'em 勝率計算アプリ（3実装）

テキサスホールデムの勝率をモンテカルロで計算するアプリを、3種類のスタックで別フォルダに用意しています。

**いずれもローカルに Node.js / Rust / Go 等をインストールする必要はありません。Docker だけでビルド・起動が完結します。**

## 各アプリの起動方法

各フォルダに入り、次のコマンドだけで起動します。

| フォルダ | コマンド | アクセス先 |
|----------|----------|------------|
| **nextjs/** | `cd nextjs && docker compose up --build` | http://localhost:3010 |
| **rust-react/** | `cd rust-react && docker compose up --build` | http://localhost:3011 |
| **go-vue/** | `cd go-vue && docker compose up --build` | http://localhost:3012 |

- 初回は `--build` でイメージをビルドします（数分かかることがあります）。
- 2回目以降は `docker compose up` で起動できます。
- 停止は `docker compose down` です。

## プリフロップ固定表（任意・3実装共通）

- **ボード0枚**のとき、169ハンドクラス×人数2〜10の固定表で即時に勝率を返します（表が生成済みの場合）。
- 固定表は **起動時には自動生成されません**。未生成でボード0枚をリクエストすると、通常のモンテカルロ計算にフォールバックして結果を返します。
- 生成手順は各フォルダの README を参照してください。

## 注意

- ポート 3000～3005 は使用していません（3010 / 3011 / 3012 を使用）。
- 各フォルダの README に API の curl 例や UI 操作手順があります。
